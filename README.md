# cv2-doc v0.0.0

cv2-doc renders documentation renders documentation generated by `idilic document`.

cv2-doc can be hosted in github pages.

cv2-doc is **serverless**.

## Getting started

```bash
$ npm i cv2-doc
$ npm i brunch
$ brunch w -s
```
